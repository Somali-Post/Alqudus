# Codex Instructions

## Project context

This project is for Alqudus Express Trucking LLC, a trucking and dispatch business based in Saint Cloud, Minnesota.

Alqudus Express focuses on owner-operators. Drivers must bring their own trucks. The company helps qualified drivers connect with contracts, jobs, and load opportunities, and Alqudus earns commission from that work.

## Build direction

- Build with Vite, React, and Tailwind CSS.
- Phase 1 is a scalable prototype: public website plus driver application form.
- Use placeholder contact details for now.
- Keep the code clean, component-based, and scalable.
- Structure the project so future backend integration is straightforward.

## Product direction

The desktop experience should feel like a professional American trucking and logistics website: clear, trustworthy, direct, and built around owner-operator recruitment.

The mobile experience should feel like a native mobile app, not a squeezed desktop website. Use large touch targets, focused sections, simple navigation, and app-style interaction patterns.

## Brand direction

- Use the circular company logo in hero, footer, and brand sections.
- Use a temporary text-based navbar logo: `ALQUDUS EXPRESS`.
- Keep messaging clear that drivers provide their own trucks.
- Do not imply that Alqudus provides trucks to drivers.

## Future scalability

The codebase should be prepared for future modules such as:

- Admin dashboard
- Driver portal
- Job management
- Document management
- Commission tracking
- Driver application review
- Load and contract tracking
