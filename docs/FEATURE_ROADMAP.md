# Alqudus Express Trucking LLC — Feature Roadmap

## Phase 1 — Website and application form

Goal: launch quickly and cheaply.

Features:

- Professional homepage
- Owner-operator recruitment page
- Driver application form
- Contact page
- Mobile-friendly layout
- Brand colors and logo

Recommended first tagline:

**Bring your truck. We help bring the work.**

## Phase 2 — Driver management dashboard

Goal: organize applications and driver records.

Features:

- Admin login
- View driver applications
- Application statuses
- Driver profile records
- Notes per driver
- Basic search/filter
- Export driver list

Suggested statuses:

- New
- Contacted
- Documents requested
- Approved
- Rejected
- Active
- Inactive

## Phase 3 — Document management

Goal: track driver compliance documents.

Features:

- Upload CDL
- Upload insurance
- Upload truck registration
- Upload W-9
- Upload medical card
- Store document expiry dates
- Expiring soon alerts
- Expired document warnings

Status labels:

- Valid
- Expiring soon
- Expired
- Missing

## Phase 4 — Job/load assignment

Goal: manage job opportunities and driver assignments.

Features:

- Create job/load
- Pickup location
- Delivery location
- Pickup date/time
- Delivery date/time
- Load value
- Assign driver
- Job status
- Notes
- Proof of delivery upload

Job statuses:

- Open
- Assigned
- Accepted
- Picked up
- In transit
- Delivered
- Completed
- Paid

## Phase 5 — Commission and payout tracking

Goal: track how the company earns commission.

Features:

- Gross load value
- Commission percentage
- Company commission amount
- Driver payout
- Deductions
- Payment status
- Monthly commission report
- Driver payout report

## Phase 6 — Driver portal

Goal: allow drivers to access job and document information.

Features:

- Driver login
- View assigned jobs
- Accept/decline jobs
- Upload proof of delivery
- Upload documents
- View payout history
- Update availability
- Update truck/trailer details

## Phase 7 — Notifications

Goal: improve communication with drivers.

Notification types:

- Application received
- Application approved
- Documents missing
- Documents expiring soon
- New job assigned
- Payment ready
- General announcement

Channels:

- Email
- SMS
- WhatsApp, if practical and budget allows

## Phase 8 — Customer/shipper lead form

Goal: allow customers to request trucking services.

Features:

- Quote request form
- Pickup city/state
- Delivery city/state
- Freight type
- Weight
- Truck/trailer needed
- Pickup date
- Contact details

## Recommended build order

1. Website + application form
2. Admin application dashboard
3. Document tracking
4. Job assignment
5. Commission/payout tracking
6. Driver portal
7. Notifications
8. Shipper/customer quote system
